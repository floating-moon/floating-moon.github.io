【随笔】Shining Memory

发表于 2024-10-11 分类于 随笔

唯有你曾与我美好的片段

像钻石一样在脑海中闪闪发光

好多同学最近都回初中校看老师了，

我激动地也闪过了这样的念头，

但马上就想起学校里的老师好像没几个看我顺眼的，

这就是刺头的下场吧。

也经常看到很多同学发些怀念初中生活的文案。

平心而论，我的初三没给我留下什么美好的回忆，每天的生活除了和老师吵架写检讨就是上课睡觉。

记得停课刚回来的时候每天和隔壁桌老哥上语文课不是聊些神秘话题就是在桌子底下动手动脚，

后来因为不写作业被老师抓去谈话，美其名曰端正学习态度，在办公室和她吵了一架，端正未遂，

但可能是和哥们日久生情激发了某种潜能，

二检语文我考了班一，他是班二，老师遂无话可说。

但在当下的迷茫中，过去的回忆却总是闪着光的，

前几天和哥们讨论关于校园恋爱的问题，

我的观点是在学校有个喜欢的人是必要的，

有时候学习的动机就是非常不纯，甚至带点幼稚，

比如我就为了能听到在意的人的一两句夸奖拼死命考了几个年段第一，

也会和数学老师赌一顿饭看数学能不能考满分，

现在想起，除了脑子会突然激动一下，内心便毫无波澜了，

这就是“贤者模式”吧。

前几天看脱口秀节目，说人生要充满 Passion，

我激动了一下，从沙发上站了起来，遂腿麻，坐了回去，

上次有 Passion 的感觉，还是在暑假在重庆集训，

晚上八点从宿舍里出来，在楼下的自动贩卖机买了瓶橘子汽水，

转首瞬间，橘霞落满天，

那是我见过最有氛围感的晚霞，

像星火燃尽最后一瞬迸出的花，

映着球场，似乎也映着我的梦，

于是按下快门，记录这一瞬间，

遗憾的是没有那个陪我看晚霞的人，

但激情不减。
