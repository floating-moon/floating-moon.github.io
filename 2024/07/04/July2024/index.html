A

Link

非常具有人类智慧的 DP。

首先有一个朴素的 DP 就是记 \(f_{i, S}\) 表示前 \(i\) 个数选的集合为 \(S\) 的方案数，然后我们发现我们并不关心选的每个数是什么，我们只关心后缀值域上每个间隔内有多少数，具体而言，假设我们考虑到前 \(i\) 个数，后 \(n - i\) 个数排序后的序列记为 \(b_1, \cdots, b_k\)，那么我们只关心每个 \((b_i, b_{i + 1})\) 中有多少个数，这样总状态数是 \(\prod_i(b_i - b_{i - 1})\)（记 \(b_0 = 0\)，\(b_{k + 1} = n + 1\)），求个导发现这个东西不大于 \(3^{n/3}\)，于是就把原来的 DP 优化为 \(O(n^23^{n/3})\)，可以通过。

B

给定整数 \(n, m\) 与 \(n\) 对整数 \((c_i, a_i)\)，其中 \(c_i = \pm1\)，求有多少正整数 \(x\) 使得 \(\sum_{i = 0}^{m - 1}x^i\) 整除 \(\sum_{i = 1}^n c_ix^{a_i}\) 或判断无穷组解。

\(n \le 10^5, 0 \le m, a_i \le 10^9\)

让两个式子同乘 \(x - 1\)，这样左边变为 \(x^m - 1\)，这样我们就可以得到 \(x^a \equiv x^{a \bmod x} \pmod{x^m - 1}\)，所以我们可以让右侧多项式的次数 \(< m\)，然后我们发现若 \(x > n\) 左边不可能整除右边（特判右侧为 \(0\)），然后我们考虑枚举 \(x\) 判断是否合法。

考虑一件神奇的事，若 \(S = \sum_{i =0}^{m - 1}c_ix^i\)，其中 \(|c_i| < x\)，则 \(x^m - 1\) 能整除 \(x\) 当且仅当 \(\forall i, c_i = 0\) 或 \(\forall i, c_i = x - 1\) 或 \(\forall i, c_i = 1 - x\)，这是因为由于 \(|c_i| < x\)，\(1 - x^m \le S \le x^m - 1\)，所以整除只可能是 \(S = 0\)，\(S = x^m - 1\)，\(S = 1 - x^m\) ，这样就能得出上述结论。

所以我们只要将右式转化为 \(S\) 的形式进行判断即可，具体而言，我们每次找到一个 \(|c_i| \ge x\)，然后将 \(c_i\) 变化至合法范围内，更新 \(c_{(i + 1) \bmod m}\) 即可，由于每次 \(c_i\) 的绝对值和减少了 \(x - 1\)，所以总共进行调和级数次操作，再加上每次取出绝对值最大的 \(c_i\)，时间复杂度 \(O(n\log^2n)\)。

CF1987F1&F2

我们考虑一个最简单的 dp，即记 \(f_i\) 表示 \([1, i]\) 最多消几次，然后我们的问题就转变为如何判断一个区间 \([l, r]\) 能被消除。

不难发现一个位置 \(i\) 能被消掉当且仅当前面消了大于等于 \((i - a_i) / 2\) 次，于是我们发现 \([l, r]\) 能被消掉的条件是 \([1, l - 1]\) 操作的次数大于等于某个值，所以记 \(g_{l, r}\) 表示 \([l, r]\) 被消除，\([1, l - 1]\) 的最少操作次数，转移很简单，第一种是枚举 \(k\)，那么 \(g_{l, r} = \max(g_{l, k}, g_{k + 1, r} - (k - l + 1) / 2)\)，后边减去的原因是因为左半边消了 \((k - l + 1) / 2\) 次，另一种情况是先消了 \([l + 1, r - 1]\)，最后消除 \(a_l, a_r\)，此时有限制 \(g_{l + 1, r - 1} \le (l - a_l) / 2\)，于是 \(O(n^3)\) 做完了。

CF1989F

学到了一个很深刻的 tarjan 分治小技巧。

不难发现每个颜色相当于限制了行和列的染色顺序，所以我们可以以此建立出一个有向图，而有向图的每一个 SCC 都表示这些点必须同时染色，所以答案为图上所有大于 \(1\) 的 SCC 的平方和。

我们怎么维护这个信息呢，考虑按时间分治，记 \(\text{solve}(l, r, E)\) 表示处理 \([l, r]\) 区间内的询问，有用的边集为 \(E\) 时的分治函数，我们首先令 \(m = (l + r) / 2\)，然后将 \(E\) 中加入时间 \(\le m\) 的边连起来跑 tarjan，此时跑出来的 SCC，在后面的时间里都存在，所以我们可以用并查集将其合并，此时若一条边的时间戳 \(> m\) 或这条边的两端仍不属于同一个 SCC，说明其可能会对后面产生贡献，将其加入 \(E'\)，然后执行 \(\text{solve}(m + 1, r, E')\)，然后用类似线段树分治的可撤销并查集撤销当前层的合并，最后执行 \(\text{solve}(l, m, E \backslash E')\) 即可，统计答案的过程在并查集的合并与撤销中体现，注意只要对在 \(E\) 中的点跑 tarjan，这样可以保证 tarjan 的时间复杂度是 \(O(|E|)\) 的，这样时间复杂度 \(O(q\log ^2q)\)。

A

给定一颗 \(n\) 个点的树，其中每个点的儿子不超过 \(d\) 个，一个排列 \(p\) 合法当且仅当对于树上任意一点，其子树在排列为一段区间，给定一个排列 \(p\)，求最少通过几次邻项交换使得其变为一个合法排列 \(q\)，以及对应 \(q\) 的个数。

\(n \le 5\times 10^4, d \le 10\)

我们让 \(p\) 变为自身的逆排列，那么问题就变为了求一个合法排列 \(q\) 使得对应排列 \(P_i = p_{q_i}\) 的逆序对最少，我们考虑一个点 \(u\)，不难发现 \(u\) 的子树是 \(u\) 的儿子的排列和 \(u\) 自己拼起来，一共只有 \(d + 1\) 个元素，我们考虑状压 dp，记 \(f_{S}\) 已经放了集合 \(S\) 中的数，所对应的 dp 值，记 \(c_{S, i}\) 表示已经放了 \(S\)，在后面添加一段 \(i\) 增加的逆序对数，那么我们有转移 \(f_{S \cup j} \gets f_{S} + c_{S,j}\)，注意转移时方案要乘上子树的方案，接下来就变为了快速计算 \(c_{S, i}\)，我们可以先计算 \(g_{i, j}\) 表示 \(j\) 拼在 \(i\) 后面产生的逆序对数，由于子树实质上是 dfn 上的一段连续区间，所以可以枚举子树小的那部分，用主席树计算贡献，这样做是 \(O(nd\log^2n)\) 的，然后用 lowbit 的技巧转移 \(c_{S, i}\)，总的时间复杂度 \(O(nd\log^2n + n2^d)\)。

CF1034E

很神奇的题目。考虑 OR 卷积会多算的东西两个有交集合产生的贡献，所以我们考虑把这一部分消掉，也就是让其成为 \(4\) 的倍数，有交等价于 \(|S| + |T| - |S\cup T| > 0\)，而 \(|S \cup T|\) 正好是 OR 卷积对应下标的 popcount，所以我们给每个数乘上一个 \(4^{|S|}\) 的权，进行一遍 OR 卷积，最后在对应位置除掉 \(4^{|S|}\) 即可把多余贡献消除，时间复杂度 \(O(n2^n)\)。
